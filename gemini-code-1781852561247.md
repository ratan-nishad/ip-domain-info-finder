# 🌐 IP & Domain Information Finder

Ye ek badhiya Python tool hai jo kisi bhi IP address ya Website (Domain) ki poori jaankari nikal leta hai.

## ✨ Khaas Features:
* **📍 IP Info:** Location, ISP (internet provider), aur country pata karein.
* **🔍 WHOIS Info:** Website kab bani aur kab expire hogi, ye dekhein.
* **🔒 SSL Info:** Website ka security certificate kab tak valid hai.
* **🕸️ Webpage Links:** Website par maujood Facebook, Instagram, Email aur Phone numbers dhoondhein.

## 🚀 Chalane Ka Tarika:
1. Sabse pehle computer me dependencies install karein: `pip install -r requirements.txt`
2. Terminal me chalane ke liye: `python3 ip_domain_info.py`
3. Browser (Web) me chalane ke liye: `python3 app.py` aur fir open karein `http://127.0.0.1:5000`
